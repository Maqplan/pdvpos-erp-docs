# Introdução

Conheça os conceitos fundamentais do ConnectTEF antes de iniciar sua integração.

---

## Objetivo

Esta seção apresenta uma visão geral da plataforma ConnectTEF, sua arquitetura, seu propósito e os conceitos necessários para compreender como ela simplifica a integração entre sistemas de gestão e o ecossistema de pagamentos.

Antes de iniciar qualquer integração, é importante entender como a plataforma foi projetada e quais problemas ela resolve.

---

## O que você aprenderá

Ao concluir esta seção você será capaz de compreender:

- O que é o ConnectTEF.
- Quais problemas a plataforma resolve.
- Como funciona sua arquitetura.
- Como o ConnectTEF se posiciona entre o ERP e os SmartPOS.
- Quais são os principais componentes da plataforma.
- Como iniciar uma nova integração.

---

## Por que começar por esta seção?

Muitos desenvolvedores iniciam diretamente pela documentação das APIs.

Embora isso seja possível, compreender a arquitetura da plataforma permite desenvolver integrações mais simples, sustentáveis e preparadas para futuras evoluções.

O ConnectTEF foi projetado para desacoplar o sistema de gestão da evolução do ecossistema de pagamentos.

Ao entender esse conceito, torna-se mais fácil compreender as decisões de arquitetura apresentadas nos capítulos seguintes.

---
